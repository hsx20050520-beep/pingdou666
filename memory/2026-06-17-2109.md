## 真正导致你连不上网关的原因

**就是这个 `netsh interface portproxy`**。我在设置局域网访问时加了 Windows 端口转发规则。这些规则**不会自动清除**，当你换了网络（手机热点），旧的转发规则还在，可能会：
- 让 Windows 网络栈混乱

## 怎么修复

在 **Windows PowerShell（管理员）** 跑一下，清除端口转发规则：

```powershell

```

然后重启 WSL：
```powershell
wsl --shutdown
```

等10秒再开 WSL 终端，网关就会恢复正常监听 localhost。
