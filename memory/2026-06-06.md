## D:\store 项目启动

应何的要求，启动 D:\store 下的两个项目：

### 项目概况
- **鲜优超市** — Next.js 16 + shadcn/ui + Supabase 全栈项目
- 同一份代码库通过 `coze` (扣子编程 CLI) 生成，用户端和商家后台是两个独立的工程项目

### 项目结构
| 目录 | 项目 | 端口 |
|------|------|------|
| `D:\store\project_20260602_205909` | 用户端（消费者App）— 移动端优先，底部 Tab 导航（首页/分类/购物车/订单） | 5000 |
| `D:\store\project_20260603_230628` | 商家后台 — 桌面端优先，左侧侧边栏 + 管理员 CRUD | 5001 |

### 两个项目特点
- **用户端**：output: 'export'，图片 unoptimized，consumer-facing
- **商家后台**：支持远程图片模式，包含 admin 路由和仪表盘

### 启动方式
- `pnpm tsx src/server.ts` （自定义 Next.js 服务器）
- 默认端口冲突 5000，用 `PORT` 环境变量区分
- 注意 `.next/dev/lock` 导致锁冲突时需要先删除 lock 文件再启动

### 访问地址
- 用户端: http://localhost:5000
- 商家后台: http://localhost:5001
